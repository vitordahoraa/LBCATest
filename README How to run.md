### Rodar todo o projeto:
Na pasta pai do projeto (Helpinho), rodar no CMD o comando:
'npm install'.

Ele irá instalar todas as dependências.

Quando terminar, basta rodar
'npm run dev'.

Tanto o servidor front-end como back-end irão começar.
Depois é só seguir na rota http://localhost:4200/ para abrir o site e começar a testar.

Obrigado pela atenção, espero que gostem \0/
